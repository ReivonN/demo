How is a webpage downloaded?

A web page begins loading as soon as the user puts in the initial request by typing in a URL, clicking a link, submitting a form or other actions. The server then begins to process the userâ€™s request. The response start is then initiated when the request is processed and sends an HTML response back across the network to the user. The DOM(Document Object Model) then starts processing and when it finishes, the user will see the web page begin rendering.

This can be seen by using the browser's network console when loading a web page.

The image above is the network console for the VMware Horizon web page for UOW remote computer labs with a total of 17 requests. We can see that the index.html is one of the first responses which indicates the response start. It is then followed by javascript, css styles and json files and lastly, the images which fill up the web page.
